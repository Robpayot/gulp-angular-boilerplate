# gulp-angular-boilerplate
A boilerplate made for quickly set up your Angular app using gulp as build system.

This is a work in progress, next stapes are compiling less and use only JS instead of coffee.js.

Requirements :

node.js
bower.js
compass
gulp

Installation :

```bash
$ npm install
$ npm bower install
```

Run :

```bash
$ gulp build
$ gulp
```
